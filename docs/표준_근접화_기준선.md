- 공통 계약 단일 기준: `/Users/iggyu/mindlog-workspace/docs/표준_근접화_기준선.md`
- 이 문서는 웹 모듈(`mindlog`) 특화 보완 항목만 다룹니다.

## 2. 웹 모듈 특화 잠금값
1. Turbo 전역 초기화는 `/Users/iggyu/mindlog-workspace/mindlog/src/main/resources/static/js/turbo-init.js` 최소 책임만 유지
2. UI 동작(알림/로딩/입력 상호작용)은 Stimulus 컨트롤러로 분리
3. 컨트롤러 성공 응답은 `303`, 검증 실패는 `422` 유지
4. 삭제 후 목록 복귀 시 `_refresh` 토큰으로 최신 목록 반영 계약 유지

## 3. 다이어리 전환 UX 메모
- 웹은 전환 판단 이벤트를 과도하게 커스텀하지 않고 Turbo 기본 방문 흐름을 따른다.
- 최소 스피너 표시 보장은 Android `VisitTransitionCoordinator`가 담당한다.

## 4. 운영 원칙
1. 공통 계약 변경은 루트 docs 먼저 수정 후 모듈 문서를 정합한다.
2. 모듈 문서는 중복 설명보다 코드 경로 중심 보완 설명을 우선한다.
